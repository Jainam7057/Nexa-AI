# Nexa AI

AI Chatbot built using the MERN Stack and Google Gemini API.

## Features

- AI Chatbot
- User Authentication
- Responsive UI

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- Gemini API
